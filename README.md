# Linear Algebra · Interactive

**Vectors, matrices, eigenvalues — the language of modern science**

**Status:** Planned

Part of [Hengzhe's Science Classroom](https://hengzhe-science-classroom.github.io/hub/) · Tier 3 · Undergraduate

## What You'll Learn

- Systems of linear equations and Gaussian elimination
- Vector spaces and subspaces
- Linear transformations and matrices
- Determinants
- Eigenvalues and eigenvectors
- Orthogonality and least squares
- Singular value decomposition

## References

- Gilbert Strang, *Introduction to Linear Algebra* (6th ed., Wellesley-Cambridge)
- Sheldon Axler, *Linear Algebra Done Right* (4th ed., Springer)
- David C. Lay, *Linear Algebra and Its Applications* (6th ed., Pearson)
- 同济大学数学系，《线性代数》第六版
